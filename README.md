# 🚀 ToDo List App

Aplicação web simples e funcional para gerenciamento de tarefas (To-Do List), permitindo criar, visualizar e excluir tarefas, com persistência de dados no navegador.

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage  

---

## ⚙️ Funcionalidades

- ✔️ Criar novas tarefas  
- ✔️ Listar tarefas dinamicamente  
- ✔️ Excluir tarefas  
- ✔️ Persistência de dados com LocalStorage  
- ✔️ Interface moderna e responsiva  

---

## 💡 Como funciona

O usuário pode adicionar uma nova tarefa através do input.  
As tarefas são armazenadas no **LocalStorage**, garantindo que não sejam perdidas ao atualizar a página.

O sistema também permite remover tarefas individualmente.
